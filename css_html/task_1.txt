1.	Опишите все известные вам способы позиционирования 5 элементов по центру, каждый из которых должен 
	занимать 20%.

Способ 1:
	элементы можно отцентрировать указав им свойство margin: 0 auto

Способ 2: 
	элементы можно отцентрировать внутри контейнера путем написания следующих свойств:

	position: relative;
    	left: 50%;
    	transform: translateX(-50%);

Способ 3:
	элементы можно отцентрировать, если задать их общему контейнеру display: flex:

	display: flex;
    	flex-direction: column;
    	align-items: center;
